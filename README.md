# Job Bot v4 Pro

Бот под схему **заказчик ↔ исполнитель** с расширенным функционалом и упором на стабильность для BotHost / VPS.

## Что внутри
- регистрация по ролям: заказчик и исполнитель
- города из твоего списка
- создание заявок: город, категория, заголовок, описание, бюджет, адрес, геолокация, фото, контакт
- модерация заявок через админку
- поиск заявок исполнителем
- отклики исполнителей
- выбор исполнителя заказчиком
- закрытие заявки
- отзывы и рейтинг
- фото к заявке
- кнопка **Построить маршрут**
- кнопка **Поделиться**
- **избранное** для исполнителя
- **жалобы** на заявку
- **поднятие заявки** наверх
- **автоархив** просроченных заявок
- **история событий** по заявке для админа
- админка: заявки, модерация, пользователи, исполнители, бан/разбан, статистика, поиск, жалобы, рассылка, экспорт БД, экспорт Excel, восстановление из DB/XLSX
- защита от типовых падений: общий error handler, антифлуд middleware, логирование в файл, SQLite WAL, безопасные настройки .env

## Для BotHost
Проект подходит под обычный polling-запуск.

## Быстрый запуск
```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
python main.py
```

## Что заполнить в .env
- `BOT_TOKEN=...`
- `ADMIN_PASSWORD=...`
- `ADMIN_TG_IDS=123456789`
- `BOT_USERNAME=your_bot_username`

## Команды
- `/start`
- `/admin`
- `/ping`
- `/cancel`

## Что можно докрутить дальше
